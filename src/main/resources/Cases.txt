Case - (1) :

(1) Ιδιοκτητης -> Update Auction
(2) User -> Bid on Auction
(3) EXCEPTION -> AuctionException.AUCTION_MODIFIED_OR_EXPIRED - [MODIFIED ΕΝ ΠΡΟΚΕΙΜΕΝΩ]
(4) Λυση : Dialog για οτι το Auction ειναι Modified ή Expired και REFRESH του Auction

#####################

Case - (2) :

(1) User -> Bid on Auction
(2) EXCEPTION -> AuctionException.AUCTION_MODIFIED_OR_EXPIRED - [EXPIRED ΕΝ ΠΡΟΚΕΙΜΕΝΩ]
(3) Λύση : Dialog για οτι το Auction ειναι Modified ή Expired και REFRESH του Auction
(4) Expired = Ηρθε το Due Date του Auction ή το Auction αγοραστηκε εφοσον εχει Buy Price
(5) Ενδεχομενως να μπορει να συγχωνευθει με το Case - (1) με ενα καταλληλο μηνυμα στο Dialog

#####################

Case - (3) :

(1) User_1 -> Bid on Auction 100 - 1st
(2) User_2 -> Bid on Auction 100 - 2nd
(2) EXCEPTION -> BidException.BID_HIGHER_BID_EXISTS στον User_2
(3) Λυση : Dialog στον User_2 οτι το Auction χρειαζεται υψηλοτερο Bid και REFRESH του Auction

#####################

Case - (4) :

(1) User -> Bid on Auction
(2) Ιδιοκτητης -> Update/Delete Auction
(3) EXCEPTION -> AuctionException.AUCTION_HAS_BID_OR_EXPIRED
(4) Λυση : Dialog για οτι το Auction εχει Bid ή ειναι Expired στον ΙΔΙΟΚΤΗΤΗ και redirect στην σελιδα που βλεπεις το auction και τα bids που αυτο εχει.

#####################

Case - (5) :

(1) Ιδιοκτητης -> Delete Auction
(2) User -> View / Bid on Auction
(3) EXCEPTION -> AuctionException.AUCTION_NOT_FOUND
(4) Λυση : Dialog για οτι το Auction δεν βρεθηκε στον User και redirect στο Home Page
